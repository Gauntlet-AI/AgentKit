Built on LangChain

Minimalist LangChain framework for building AI agents

AgentKit is an opinionated layer on top of LangChain that makes it easy to integrate custom AI agents into your applications. Define your tools and agent behavior in a simple YAML file - from chatbots to specialized AI assistants.

Get Started Star on GitHub 
# AgentKit Configuration
name: "CustomerSupportAgent"
description: "AI assistant for customer support"

tools:
  - name: "ticket_search"
    description: "Search through support tickets"
    path: "./tools/ticket_search.py"
    
  - name: "knowledge_base"
    description: "Access product documentation"
    path: "./tools/kb_access.py"
    
  - name: "email_sender"
    description: "Send response emails"
    path: "./tools/email.py"

documentation:
  - path: "./docs/support/**/*.md"
  - url: "https://api.your-product.com/docs"

models:
  default: "gpt-4"
  fallback: "gpt-3.5-turbo"
from agentkit import Agent

# Initialize your custom support agent
support_agent = Agent.from_config("support_agent_config.yml")

# Handle customer inquiry
response = support_agent.handle_inquiry("""
    Customer reported issues with login
    after recent password reset
""")

# The agent will automatically:
# 1. Search relevant support tickets
# 2. Check knowledge base for solutions
# 3. Draft and send a response email
print(response.summary)
